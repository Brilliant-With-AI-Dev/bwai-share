What's already decided

Locked-in design calls — the flow above reflects these.

  1. The prompt writes itself in front of you. No loading screen between asking and seeing — the prompt streams into the workspace as it's generated.
  2. Two prompt shapes, fixed. One for a one-off prompt, one for a reusable template. Testers can't edit those shapes — only what comes out of them.
  3. No user profile in this version. The system has slots for things like role, tone, audience, but we leave them empty for v1. Wired up in a later version.
  4. "+ New" reuses the welcome input. Same one sentence + reusable-template checkbox, just inside a small dialog.
  5. Reusable templates tell ChatGPT to ask first. When a tester copies a reusable template into ChatGPT, the template instructs ChatGPT to ask the user for the missing pieces before producing output.
  6. Sharpening questions appear once, on first generation. We stream the prompt plus up to 3 optional follow-up questions. Answering any and clicking "Apply answers" rewrites the prompt once. After that, all further edits go through the "Improve" box — no more pop-up questions.
